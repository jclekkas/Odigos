137 - SEO: Titles, Linking, FAQs & Content Depth — Priority Pages

What & Why
Google Search Console shows impressions but low CTR on several high-opportunity pages. Two combined briefs drive this task:

Brief A — Push pages into top-5 by strengthening titles/meta, targeted internal links, and FAQs where a real content gap exists.
Brief B — Push selected pages into top-3 by adding query expansion ("Common Questions" sections), direct-answer snippet blocks, one actionable competitor-gap section per page, and subtle entity/term enrichment.

Focus is strictly on these pages:
- /dealer-wont-give-otd-price (highest priority)
- /dealer-doc-fee
- /car-dealer-fees-by-state (title/meta + state links only)
- Texas, New York, California, Florida, Ohio state pages (titles/meta for all 5; deep content additions for TX, NY, CA only)

Done looks like

Titles & Meta (all priority pages)
- Every priority page has a fully rewritten title that includes the year (2026), an intent-matching keyword (doc fee, OTD, dealer fee, legal/limit language), and a clear user benefit. No generic titles.
- Target titles:
  - /dealer-wont-give-otd-price → "Dealer Won't Give an OTD Price? Here's Exactly What to Do (2026 Guide)"
  - /dealer-doc-fee → "Dealer Doc Fee (2026): What It Is and What Dealers Can Legally Charge"
  - /car-dealer-fees-by-state → "Car Dealer Fees by State (2026): Caps, Limits, and What Dealers Can Charge You"
  - TX → "Texas Dealer Doc Fees (2026): What Dealers Can Legally Charge You"
  - NY → "New York Dealer Doc Fee Cap (2026): What Dealers Are Allowed to Charge"
  - CA → "California Dealer Doc Fee (2026): $85 State Cap — What Dealers Can Legally Charge"
  - FL → "Florida Dealer Fees (2026): No State Cap — What to Watch Out For"
  - OH → "Ohio Dealer Fees (2026): What Dealers Can Legally Charge You"
- Every priority page's meta description is 140–160 chars, names the user benefit, and includes at least one target keyword.

Technical / SEO safety
- Canonical tags remain written exclusively by setSeoMeta in useEffect. Do NOT add a second canonical system or manually inject <link rel="canonical">.
- JSON-LD must be injected via Helmet using existing utilities from client/src/lib/jsonld.ts only. Do not add new schema types.
- Multiple schema blocks may coexist in one Helmet block. Do not add multiple Helmet blocks to the same component.
- Do not repeat exact same question wording across multiple pages. Questions should be adapted to each page’s topic and intent.
- Competitor-gap sections must be ≤150 words and actionable.
