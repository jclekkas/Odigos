Implement Task #135 in the existing Odigos codebase. This is an implementation task, not a planning task.

Objective
Refactor the analyze route into a thin HTTP layer backed by a service module, and remove unused auth/session dependencies.

What & Why
The analyze route at server/routes/analyze.ts is currently about 752 lines and mixes:
- HTTP handling
- AI orchestration
- state detection
- circuit-breaker logic
- warehouse logging/enqueueing

This should be refactored so the route is thin and the business logic lives under server/services/. Also, passport, passport-local, and express-session are installed but unused and must be fully removed.

Done looks like
- The analyze route handler is 150 lines or fewer
- All business logic lives in one or more dedicated service modules under server/services/
- The existing API tests for /api/analyze continue to pass without modification
- passport, passport-local, and express-session are removed from package.json and absent from the codebase
- TypeScript compiles cleanly with no new errors

Out of scope
- Adding authentication
- Adding sessions
- Changing the public API shape of /api/analyze
- Adding new tests unless strictly required to preserve existing behavior

Relevant files
- server/routes/analyze.ts
- server/routes/index.ts
- server/index.ts
- package.json

Required implementation

1. Extract an AnalyzeService
Create server/services/analyzeService.ts as the main orchestration layer for /api/analyze.

Move out of server/routes/analyze.ts:
- AI orchestration
- state detection
- circuit-breaker behavior
- warehouse enqueueing/logging
- any other non-HTTP business logic currently embedded in the route

Keep in server/routes/analyze.ts:
- request parsing
- input validation
- calling the service
- mapping the service result to the HTTP response
- preserving existing status codes and JSON response shape
- translating thrown errors into the same HTTP semantics as today

2. Remove dead auth dependencies
Remove these packages from package.json:
- passport
- passport-local
- express-session

Then search the full repo and remove any related:
- imports
- configuration
- middleware
- dead references
- unused code paths
