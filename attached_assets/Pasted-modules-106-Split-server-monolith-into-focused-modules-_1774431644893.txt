Be especially careful with: webhook parsing behavior, route-specific middleware, imported singleton state/caches, cross-module type dependencies, metrics cache behavior, BI filter normalization, health/sitemap/robots behavior.
Prefer minimal necessary import updates over broad cleanup.
Do not opportunistically rename unrelated symbols or reformat the entire codebase.
Preserve comments where useful, especially around subtle behavior.
