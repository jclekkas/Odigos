What & Why
The analyze route (server/routes/analyze.ts) is 752 lines of mixed concerns — HTTP handling, AI orchestration, state detection, circuit-breaking, and warehouse logging all live in one file. Extracting a thin service layer makes the code easier to test in isolation, reason about, and extend. Separately, passport, passport-local, and express-session are installed but never used anywhere; removing them reduces the attack surface.
